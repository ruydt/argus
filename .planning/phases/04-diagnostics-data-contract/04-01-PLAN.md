<threat_model>
Threat: diagnostics aggregation could accidentally include captured event content or sample fields.
Severity: high.
Mitigation: domain storage aggregate structs must contain only counts and timestamp fields, with no event content, command, path, prompt, diff, raw payload, stdout, stderr, or transcript sample fields.

Threat: diagnostics aggregation could become expensive on large local databases.
Severity: medium.
Mitigation: implement dedicated SQL aggregate queries using `COUNT(*)` and `MAX(created_at)` instead of loading event/session rows into Go memory.
</threat_model>
