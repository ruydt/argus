<threat_model>
Threat: a new diagnostics route could bypass existing localhost/CORS/host protections.
Severity: high.
Mitigation: mount diagnostics through the existing `server.NewRouter` mux so existing `panicRecovery`, `hostHeader`, `corsAllowlist`, and `logging` middleware wrap the endpoint.

Threat: diagnostics could expose storage or event internals beyond the locked contract.
Severity: high.
Mitigation: handler must encode only `domain.Diagnostics` and must not query or serialize event/session rows.
</threat_model>
